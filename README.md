
### 核心库

- 导航库 react-navigation 4.x 版本
- 本地存储 react-native-storage 初始化在全局中，使用方式：```storage.save | load```

### 项目结构
```

|-js
  |-assets              // 本地资源文件
  
  |-common              // 自定义组件、共用组件
  
  |-global              // 全局定义

  |-router              // 基于react-navigation 路由跳转封装 

  |-screens             // 业务组件相关

```
